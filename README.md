# Binus Flow

A simple task management app with **drag & drop** functionality to update task statuses.

---

## Live Demo

Try it here: https://web-binusflow.vercel.app

---

## Local Development

### Requirement

- Node.js

### Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/itsdomibro/web-binusflow.git
   cd web-binusflow
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open http://localhost:3000 in your browser

## Tech Stack

### React Framework

- Next.js – routing & server-side rendering
- Zustand – state management & persistence
- react-hook-form + @hookform/resolvers – form handling
- etc.

### UI Framework

- TailwindCSS
- ShadCN/UI – pre-built components

## Application Features

- **Responsive Design**
- **Task Management**
- **Color Management**
- **Drag & Drop** - update task state seamlessly with drag-and-drop interaction
- **Themes** - switch between light and dark modes
- **Collapsible Sidebar**
- **Toast Notifications** - instant feedback for user actions
- **Completion Animation** - dopamine hit when task is marked as done
